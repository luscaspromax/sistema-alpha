# ⚡ SISTEMA ALPHA 2.0 | Hub de Performance Biológica

Este é o repositório oficial do **Portal Interativo Sistema Alpha**. Uma ferramenta desenvolvida para otimização hormonal, biohacking e gestão de performance masculina.

## 🚀 Funcionalidades do Portal
- **Alpha Scan v2.0:** Algoritmo interativo que analisa sono, peso e estresse para diagnosticar a prontidão neuromuscular diária.
- **Gym Analyzer:** Recomendação dinâmica de treino (Força, Hipertrofia ou Deload) baseada no Alpha Score.
- **Protocolo 21 Dias:** Guia estruturado de reprogramação biológica dividido em fases.
- **Calculadora de Bio-Feedback:** Cálculo automático de hidratação e Taxa Metabólica Basal (TMB).

## 🛠️ Tecnologias Utilizadas

- CSS3 (Design Premium Dark/Red Aesthetic)
- JavaScript (Lógica de Diagnóstico e Manipulação de DOM)
